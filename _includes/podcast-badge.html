{%- comment -%} Renders a research-radio podcast badge for a publication, matched by DOI against the episodes collected in `_data/research-radio.json`. Parameters: publication — a record from `_data/publications.yml` (needs a `doi` field). No-ops silently when research-radio data is missing or no episode matches, so it is safe to include before any own-paper episodes exist. {%- endcomment -%} {%- assign radio_data = site.data.research-radio -%} {%- if publication.doi and radio_data and radio_data.episodesByDoi -%} {%- assign doi_key = publication.doi | downcase -%} {%- assign episode = radio_data.episodesByDoi[doi_key] -%} {%- if episode -%} Podcast{% if episode.duration %} · {{ episode.duration }}{% endif %} {%- endif -%} {%- endif -%}
